My Recent IOT Integration Mini Project

[S.M.A.R.T TAMAN(GARDEN)]

The Problem: The Garden by the Bay is a well-known tourist attraction in Singapore's southern region. While it showcases a wide variety of plants and flowers, it lacks technological attractions and cleaning efficiency, causing dissatisfaction among visitors.

The Objective: I believe with a few tweaks, we can significantly improve the visitors' experience and draw more attention, encouraging them to return with family and friends.

Type of Sensors, Actuators, Microcontroller, Communication Platform Used:

  • 1) Ultrasonic sensor (Measure the height of rubbish in the bin, notify if full)

  • 2) Light sensor (Measure light, act as a master switch for lights)

  • 3) Temperature/Humidity sensor (Notify staff if it's going to rain)

  • 4) Soil Moisture sensor (Turn on sprinkler when soil is dry)

  • 5) Relay (Used to control the sprinkler)

  • 6) 2 RGB lights (Weather indication, and Super tree grove)

  • 7) MP3 module (Play a sound, indicating the bin is full)

  • 8) Zigbee and WIFI (esp8266) module (Send data, Machine to Machine and web service respectively)

  • 9) 3 Arduino Nano Microcontroller

  • 10) 1 Raspberry Pi, Single-Board Computer
